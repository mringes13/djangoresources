Creating Django Environment

In console:

conda create --name djangoEnv python=3.7

Activate Django Environment

In console:

conda activate djangoEnv

Create Project

In console:

django-admin startproject project_test

Create Application

In console:

python manage.py startapp app_test

Run Server

In console:

python manage.py runserver

Connect Application to Project

In project settings.py: 

Installed_Apps = [
  'app_test'
]

In application views.py: 

from django.http import HttpResponse
def index(request):
  return HttpResponse("I am the index page").

Connect Application Views to Project

In project urls.py:

from application import views
urlpatterns = [
  path('', views.index, name='index')
]

Adding Template Tagging

(1) Create templates folder in line with application and project folders.

(2) Create folder in the templates folder for each application.

(3) Create an index.html file for each application.

In project settings.py:

import os
TEMPLATE_DIR = os.path.join(BASE_DIR, 'templates')

DIRS=[TEMPLATE_DIR,],

In template application index.html: 

{% load static %}
<body>
  <h1> Index Page </h1>
  {{ page_template }}
</body>>

In application views.py: 

def index(request):
  my_dict={'page_template':'This is the text that would be inserted as the template tag'}
  return render(request,'application/index.html', context=my_dict)

Connecting Models Templates Views

In application views.py:

from application.models import Model1, Model2
def index:
  modelobjects = Model1.objects.order_by('column1')
  dictionary={'modelobjects':modelobjects}
  return render(request, 'application/index.html', context=modelobjects)

In templates application index.html:

{% load static %}
<html>
  <link rel="stylesheet" href={% static 'css/mystyle.css' %}/>
  ...
  <h1>Hello!<h1>
  <h2>Here are the model objects populated.</h2>
  <div class='proj'>
    {% if modelobjects %}
      <table>
        {% for object in modelobjects %}
          <tr>
            <td>{{ object.column1 }}
          </tr>
        {% endfor %}
      </table>
    {% else %}
      <p>No objects found!</p>
    {% endif %}
  </div>
</html>

Getting Data from Forms

In application views.py:

def form_name_view(request):
  if request.method == 'POST':
    form = forms.FormName(request.POST)
    if form.is_valid():
      print("Validation Success")
      print("Name="+form.cleaned_data['name'])
      form.save(commit=True)
      return index(request)
  return render(request, 'application/form.html', {'form':form})

Clean Method Form Validation

In application forms.py class FormName:

class FormName(forms.Form):
  botcatcher = forms.CharField(required=False, widget=forms.HiddenInput)
  def clean_botcatcher(self):
    botcatcher = self.cleaned_data['botcatcher']  
    if len(botcatcher)>0:
      raise forms.ValidationError("Caught bot!")
    return botcatcher

Import Method Form Validation

In application forms.py:

from django import forms
from django.core import validators
class FormName(forms.Form):
  botcatcher=forms.CharField(required=False, widget=forms.HiddenInput, validator = [validators.MaxLengthValidator(0)])

Custom Method Form Validation

In application forms.py:

def check_for_letter(value):
  if value[0] != '_':
    raise forms.ValidationError("Field must start with __")
class FormName(forms.Form):
  field=forms.CharField(validators=[check_for_letter,])

Clean Entire Form

In application forms.py:

class FormName(forms.Form):
  email=forms.EmailField()
  verify_email=forms.EmailField(label="Enter your email again:")
  def clean(self):
    all_clean_data=super().clean()
    email=all_clean_data['email']
    vemail=all_clean_data['verify_email']
    if email!=vemail:
      raise forms.ValidationError("Make sure your emails match!")

URL Template Tagging

(1) In templates/application create multiple html files.

In application views.py:

def index(request):
  return render(request, 'app/index.html')

In project urls.py:

from application import views

In application urls.py:

from django.conf.urls import url

from application import views
app_name='application'

urlpatterns=[
  path('relative/', views.relative),
  path('other/', views.other)
]

In templates application relative.html:

<a href="{% url 'application:other' %}"> Go to other.html </a>

Template Inheritance

(1) Create folder "templatetags under application.

(2) In application/templatetags, create a __init__.py file.

(3) In application/templatetags, create a customfilters.py.

In customerfilters.py:

from Django import template
register = template.library()
@register.filter(name='cut')
def cut(value, arg):
  """
  This cuts out all args from the string
  """
  return value.replace(arg,'')
register.filter('cut', cut)
